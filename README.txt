TicTacToe


A tic-tac-toe AI that is invincible! Made by Addison Djatschenko
(AddisonPascal)

Features:

| Statistics
| Total games played
| User score
| User score percentage
| User wins
| User win percentage
| Computer wins
| Computer win percentage
| Draws
| Draw percentage
| How good you are ('you are **% as good as the computer')
| Play
| Resume game from ID
| Info
| Share position via email
| Copy ID to clipboard
| Go back to home
| Continue game as X
| Create ID from game
| Quit
